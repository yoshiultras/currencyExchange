# currencyExchanger

Простой обучающий проект в виде API обменника валют.

### Мотивация:

- Работа с голыми Java сервлетами без фреймворков.
- Работа с HTTP (заголовки, тела сообщений, коды ответа)
- Практика в написании REST архитектуры.

### Технологии:

- Java 17
- SQLite 3

Также были использованы библиотки Gson и Lombok.
